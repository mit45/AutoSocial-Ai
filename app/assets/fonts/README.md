# Fontlar (TTF)

Bu klasöre TTF font dosyalarını ekleyin. Görsel render için kullanılır.

- **main.ttf** – Ana metin ve (yoksa) imza için kullanılır.
- İsterseniz başka bir isimle TTF ekleyebilirsiniz; ilk bulunan `.ttf` kullanılır.

Örnek: Google Fonts'tan indirdiğiniz bir fontu `main.ttf` olarak buraya kopyalayın.
